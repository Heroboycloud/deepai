# 🤖 DeepSeek Terminal Chat

A beautiful, feature-rich terminal-based chat interface for OpenRouter AI models with session management, markdown rendering, and multi-model support.

![DeepSeek Terminal Chat](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-yellow)

## ✨ Features

- 🎨 **Rich Terminal Interface** - Beautiful formatting with colors, panels, and emojis
- 📝 **Markdown Support** - Renders AI responses in markdown format with syntax highlighting
- 💾 **Session Management** - Save and load multiple chat sessions
- 🤖 **Multi-Model Support** - Choose from 8 different AI models:
  - DeepSeek Chat & Coder
  - Meta Llama 3.3
  - Mistral 7B
  - Google Gemini Pro
  - Claude 3 Haiku
  - Microsoft Phi-3
  - Qwen 2.5
- ⚙️ **System Prompts** - Set custom system prompts per session
- 📜 **Conversation History** - View, clear, and manage chat history
- 🔒 **Persistent Storage** - Sessions saved locally in `~/.deepseek_chat/`
- ⌨️ **Intuitive Commands** - Easy-to-use slash commands for all features
- 🚀 **Quick Launch** - Start with a session name as command-line argument

## 📋 Prerequisites

- Python 3.8 or higher
- OpenRouter API key ([Get one here](https://openrouter.ai/))
